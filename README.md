# Description
 - **프로젝트 소개** <br>
   다양한 미니 게임을 즐길 수 있는 미니 메타버스 게임을 만들었습니다. <br>
   NPC 대화 시스템, 상점, 인벤토리 시스템, 스코어 보드, 아이템 시스템을 구현했습니다.<br><br>
 - **개발 기간** : 2025.02.14 - 2025.02.20
<br><br>
